# Dados

Esta pasta contém os datasets utilizados pela PoC.

## Estrutura prevista

- `sessoes_simuladas.csv` — sessões de carga geradas para testar a PoC (a ser produzido pela Fase 2)
- `tarifas_horarias.csv` — faixas de tarifa por horário (ponta / intermediária / fora-ponta)
- `historico/` — histórico real ou simulado de consumo da rede

## Reaproveitamento da Sprint 04

A base preparada na Sprint 04 (`fase1-base_de_dados-final.csv`) pode ser reaproveitada como histórico de sessões para alimentar a simulação. Se for esse o caso, copiar o arquivo para esta pasta e documentar a origem aqui.
